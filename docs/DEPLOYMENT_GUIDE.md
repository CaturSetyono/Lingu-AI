#### 3.3 Connect to Vercel

1. Go to [https://vercel.com/dashboard](https://vercel.com/dashboard)

3. Select your GitHub repo (`ai-teksgen`)
4. Click **Import**

   - Key: `VITE_OPENROUTER_API_KEY`
   - Value: `sk_live_xxxxxxxxxxxxx`
6. Click **Deploy**

Vercel will:

- Auto-detect Astro
- Build with `npm run build`
- Deploy to live URL (usually `https://ai-teksgen.vercel.app`)

### 7. Troubleshooting

#### Issue: "API key not configured"

**Solution:** Add `VITE_OPENROUTER_API_KEY` to `.env` (local) or Vercel dashboard

#### Issue: "Build fails with Tailwind error"

**Solution:** Already fixed - ensure you're on latest commit. Run `npm run build` to verify

#### Issue: PWA doesn't install

**Solution:**

- Check manifest.json exists in public/
- Verify HTTPS (Vercel provides this automatically)
- Chrome DevTools → Application → Manifest should show no errors

#### Issue: "Qwen model rate limited"

**Phase 2:** Add retry logic + fallback model (Claude, GPT)

#### Issue: Component styling looks wrong

**Solution:** Neo-Brutalism is bold/high-contrast by design:

- Check browser zoom (should be 100%)
- Clear cache: Hard refresh (Ctrl+Shift+R)
- Check .css files loaded (DevTools → Network)
